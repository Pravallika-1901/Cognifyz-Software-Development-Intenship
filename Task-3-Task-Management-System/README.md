# Smart Task Management System

## Description
A menu-driven Java application to manage daily tasks using CRUD (Create, Read, Update, Delete) operations.

## Features
- Add New Task
- View Tasks
- Update Task
- Delete Task
- Search Task
- Menu-driven interface
- Input validation

## Technologies Used
- Java
- ArrayList
- Scanner
- Methods
- Loops
- Switch Case

## Concepts Covered
- CRUD Operations
- Collections (ArrayList)
- User Input Handling
- Methods
- Conditional Statements
