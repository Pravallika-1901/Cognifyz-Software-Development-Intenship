# 🌡️ Smart Temperature Converter

## 📌 Overview

The Smart Temperature Converter is a Java console application developed as part of the Cognifyz Technologies Software Development Internship. It enables users to convert temperatures between Celsius, Fahrenheit, and Kelvin through a simple menu-driven interface. The application provides accurate conversions and allows users to perform multiple operations until they choose to exit.

---

## 🎯 Task Objective

The objective of this task is to develop a temperature conversion application that:

- Accepts temperature values from the user.
- Supports multiple temperature scales.
- Converts temperatures accurately between Celsius, Fahrenheit, and Kelvin.
- Demonstrates the use of methods, loops, switch-case statements, and user input handling in Java.

---

## 🚀 Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Convert Celsius to Kelvin
- Convert Kelvin to Celsius
- Convert Fahrenheit to Kelvin
- Convert Kelvin to Fahrenheit
- Interactive menu-driven interface
- Continuous execution until the user exits
- Simple and user-friendly console application

## ▶️ How to Run

1. Open the project in any Java IDE (VS Code, IntelliJ IDEA, Eclipse, etc.).
2. Compile the Java source file.
3. Run `TemperatureConverter.java`.
4. Select the desired conversion option.
5. Enter the temperature value.
6. View the converted result.
7. Continue converting temperatures or exit the application.
